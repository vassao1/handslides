# Handslides
Estudo/PoC de OpenCV utilizando o framework Mediapipe pra detecção de mãos.
O objetivo é criar um controle pra slides utilizando gestos, se você levantar um dedo, ele passa o slide, se você levantar dois dedos, ele volta o slide.
A lógica de "dedos levantados"/gestos fica no arquivo gestos.py, então dá pra inventar bastante coisa diferente, passar como valor da função e só interpretar dentro da main e utilizar o pyautogui pra automatizar inputs, esse código é somente um proof of concept :)

## Como utilizar:
- Clona o repositório
- Cria o .venv
- Instala os requirements via pip
- Roda o main.py
